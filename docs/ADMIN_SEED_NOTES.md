# Admin Seed Instructions

1. اطمینان حاصل کنید که متغیرهای محیطی پایگاه داده تنظیم شده‌اند (`MYSQL_DSN`، در صورت نیاز `DEV_ADMIN_PASS`).
2. برای اجرای اسکریپت بذرگیری دستور زیر را در ریشه مخزن اجرا کنید:
   ```bash
   node sql/seed_admins.js
   ```
3. در صورت تنظیم `DEV_ADMIN_PASS` همان رمز عبور برای کاربر `Coaxys` استفاده می‌شود. در غیر این صورت اسکریپت یک رمز عبور جدید تولید کرده و در خروجی ترمینال با پیام «DEVELOPMENT ONLY» چاپ می‌کند.
4. رمزهای عبور چاپ شده را صرفاً برای محیط‌های توسعه نگه دارید و بلافاصله پس از استقرار در محیط عملیاتی آن‌ها را تغییر دهید.
5. نقش‌ها و کاربران ایجاد شده:
   - `Coaxys` با نقش `absolute_developer`
   - `school_team_1` با نقش `school_team`
   - `dev_team_1` با نقش `developer_team`
6. اگر نیاز به بازیابی رمز عبور توسعه داشتید، اسکریپت را دوباره اجرا کنید یا `DEV_ADMIN_PASS` را قبل از اجرا تنظیم کنید.
